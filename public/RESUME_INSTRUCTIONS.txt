RESUME SETUP INSTRUCTIONS
=========================

To make the Resume button work on your portfolio:

1. Create or export your resume as a PDF file
2. Name the file: resume.pdf
3. Place it in this folder (public/)
4. The Resume button in the navbar will automatically link to it

The button will open your resume in a new tab when clicked.

File path should be: public/resume.pdf

Note: Make sure the PDF is professional and up-to-date!
