## 🎨 Color Usage Guide

### When to Use Each Color

**Primary Blue (#0ea5e9)**
- Main CTAs and buttons
- Links and interactive elements
- Active states
- Primary headings

**Secondary Cyan (#06b6d4)**
- Gradient combinations with primary
- Secondary buttons
- Hover states
- Accent elements

**Accent Green (#10b981)**
- Success messages
- Availability indicators
- Positive actions
- Checkmarks and confirmations

**Gradients**
- Hero sections: `linear-gradient(135deg, primary, secondary)`
- Buttons: `linear-gradient(135deg, primary, primary-dark)`
- Borders: `linear-gradient(90deg, primary, secondary)`
